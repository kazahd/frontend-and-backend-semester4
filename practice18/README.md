# Приложение для управления задачами (PWA)

## Описание проекта

Веб-приложение для управления списком задач с поддержкой:
- Офлайн-режима
- Push-уведомлений
- Напоминаний с возможностью отложить
- Установки на устройство (PWA)

## Функциональные требования

### Практическое занятие №13 - Service Worker
- [x] Регистрация Service Worker
- [x] Кэширование статических ресурсов
- [x] Работа в офлайн-режиме
- [x] Сохранение задач в localStorage

### Практическое занятие №14 - Web App Manifest
- [x] Файл manifest.json со всеми полями
- [x] Иконки разных размеров (PNG)
- [x] Мета-теги для iOS и Android
- [x] Кнопка установки приложения

### Практическое занятие №15 - HTTPS + App Shell
- [x] Локальный HTTPS через mkcert
- [x] App Shell архитектура
- [x] Кэширование статики (Cache First)
- [x] Динамический контент (Network First)

### Практическое занятие №16 - WebSocket + Push
- [x] Сервер на Node.js с Socket.IO
- [x] Отправка и получение событий в реальном времени
- [x] Push-уведомления через VAPID
- [x] Подписка/отписка на уведомления

### Практическое занятие №17 - Детализация Push
- [x] Форма добавления напоминаний с датой/временем
- [x] Планирование push-уведомлений на сервере
- [x] Кнопка "Отложить на 5 минут"
- [x] Уведомление об откладывании

## Технологии

- HTML5, CSS3, JavaScript (ES6+)
- Service Worker API
- Web App Manifest
- WebSocket (Socket.IO)
- Push API / Web Push
