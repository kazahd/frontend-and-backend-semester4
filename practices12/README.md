# Интернет-магазин: Веб-приложение на React + Express

## О проекте

Полноценное веб-приложение интернет-магазина с системой аутентификации, управлением товарами и ролевой моделью доступа (RBAC). Разработано в рамках дисциплины "Фронтенд и бэкенд разработка".

## Технологии

**Бэкенд:** Node.js, Express.js, bcrypt, JWT, nanoid, CORS, Swagger

**Фронтенд:** React, Axios, Sass (SCSS), React Hooks

**Документация:** Swagger UI


## Выполненные практические занятия

### Практическое занятие №7. Basic Auth и bcrypt
Реализованы маршруты регистрации и входа пользователей. Пароли хешируются с помощью bcrypt с использованием соли для повышения безопасности.

### Практическое занятие №8. JWT-токены
Добавлена выдача access-токенов при входе в систему. Создана middleware для проверки токенов и защищённый маршрут получения информации о текущем пользователе.

### Практическое занятие №9. Refresh-токены
Реализована генерация refresh-токенов вместе с access-токенами. Добавлен эндпоинт для обновления пары токенов с ротацией refresh-токенов.

### Практическое занятие №10. Хранение токенов на фронтенде
Токены хранятся в localStorage. Настроены перехватчики Axios для автоматической подстановки токенов в заголовки и обновления токенов при ошибке 401.

### Практическое занятие №11. Управление доступом на основе ролей (RBAC)
Внедрена система ролей: гость, пользователь, продавец, администратор. Создана middleware для проверки прав доступа, настроены права для каждого эндпоинта согласно заданию.

### Практическое занятие №12. Подготовка к контрольной работе
Выполнено тестирование работоспособности приложения. Подготовлен README.md с описанием проекта и инструкцией по запуску.
